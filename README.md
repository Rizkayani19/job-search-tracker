# 🚀 Cloud-Sync Job Search CRM

A full-stack, serverless web application designed to automate and organize the job application process. Built with a focus on mobile-first utility and real-time data synchronization.

## 🌐 Live Demo
[Click here to view my app!](https://rizkayani19.github.io/job-search-tracker/)

## ✨ Key Features
- **Cloud Infrastructure:** Real-time data sync across all devices using **Firebase**.
- **Secure Auth:** Google OAuth integration with mobile-optimized redirect handling.
- **Smart Tracking:** Color-coded "Follow-up" badges (Green/Orange/Red) based on application aging.
- **Resume Management:** Capability to store and retrieve PDF/Word resumes via Base64 database encoding.
- **One-Click Automation:**
  - **Research:** Instantly opens LinkedIn, Glassdoor, and Google News for specific companies.
  - **Follow-up:** Auto-generates professional email drafts using `mailto:` protocols.
  - **Sharing:** Native mobile sharing integration via the Web Share API.
- **Data Portability:** Export entire job history to **CSV/Excel** with one click.

## 🛠️ Built With
- **Frontend:** HTML5, CSS3 (with Dark Mode support), JavaScript (ES6 Modules)
- **Backend-as-a-Service:** Firebase (Auth & Realtime Database)
- **Deployment:** GitHub Pages

## 💡 How to Use
1. **Login:** Use your Google account to unlock your private cloud database.
2. **Add Job:** Fill in the company details and upload your resume.
3. **Track:** Monitor your stats (Applied vs. Interviews) and average salary.
4. **Action:** Use the "Research" or "Email" buttons to stay ahead of the competition.
